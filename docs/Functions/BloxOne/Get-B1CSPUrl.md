# Get-B1CSPUrl

## SYNOPSIS
Retrieves the stored BloxOneDDI CSP Url, if available.

## SYNTAX

```
Get-B1CSPUrl
```

## DESCRIPTION
This function will retrieve the saved BloxOneDDI CSP Url from the local user/machine if it has previously been set.

## EXAMPLES

### EXAMPLE 1
```powershell
Get-B1CSPUrl
```
