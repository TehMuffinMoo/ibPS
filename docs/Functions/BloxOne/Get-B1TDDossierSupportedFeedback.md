# Get-B1TDDossierSupportedFeedback

## SYNOPSIS
Queries a list of available dossier feedback types

## SYNTAX

```
Get-B1TDDossierSupportedFeedback
```

## DESCRIPTION
The Dossier Indicators cmdlet returns a list of all Dossier (sometimes referred to as Intel Lookup) feedback types.

## EXAMPLES

### EXAMPLE 1
```
Get-B1TDDossierSupportedFeedback
```
