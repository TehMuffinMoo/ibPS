# Get-B1DHCPGlobalConfig

## SYNOPSIS
Retrieves the BloxOneDDI Global DHCP Configuration

## SYNTAX

```
Get-B1DHCPGlobalConfig [[-Fields] <String[]>]
```

## DESCRIPTION
This function is used to retrieve the BloxOneDDI Global DHCP Configuration

## EXAMPLES

### EXAMPLE 1
```
Get-B1DHCPGlobalConfig
```
