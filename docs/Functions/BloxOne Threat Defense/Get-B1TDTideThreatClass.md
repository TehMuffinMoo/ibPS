# Get-B1TDTideThreatClass

## SYNOPSIS
Queries a list of threat classes

## SYNTAX

```
Get-B1TDTideThreatClass [[-id] <String>]
```

## DESCRIPTION
This function will query a list of threat classes

## EXAMPLES

### EXAMPLE 1
```powershell
Get-B1TDTideThreatClass -id "Bot"
```
