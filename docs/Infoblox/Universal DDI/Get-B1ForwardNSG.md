# Get-B1ForwardNSG

## SYNOPSIS
Retrieves a list of Forward DNS Server Groups from BloxOneDDI

## SYNTAX

```
Get-B1ForwardNSG [[-Name] <String>] [-Strict] [[-Limit] <Int32>] [[-Offset] <Int32>] [[-tfilter] <String>]
 [[-Fields] <String[]>] [[-OrderBy] <String>] [[-OrderByTag] <String>] [[-CustomFilters] <Object>]
 [[-id] <String>] [<CommonParameters>]
```

## DESCRIPTION
This function is used to query a list Forward DNS Server Groups from BloxOneDDI

## EXAMPLES

### EXAMPLE 1
```powershell
Get-B1ForwardNSG -Name "Data Centre" -Strict
```
