# Get-B1Applications

## SYNOPSIS
Retrieves a list of supported Infoblox Portal Applications

## SYNTAX

```
Get-B1Applications [<CommonParameters>]
```

## DESCRIPTION
This function is used query a list of supported Infoblox Portal Applications/Services

## EXAMPLES

### EXAMPLE 1
```powershell
Get-B1Applications

dfp
dns
dhcp
cdc
anycast
orpheus
msad
authn
ntp
rip
bgp
ospf
discovery
```
