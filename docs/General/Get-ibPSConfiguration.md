# Get-ibPSConfiguration

## SYNOPSIS
Used to get the current configuration for ibPS

## SYNTAX

```
Get-ibPSConfiguration [-IncludeAPIKey]
```

## DESCRIPTION
This function is used to get the current configuration for ibPS
